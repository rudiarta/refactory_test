## How To Use
First install sql-migrate
following:
```sh
go get -v github.com/rubenv/sql-migrate/...
```
<br>
Config the `dbconfig.yml` and the `.env` file , to you mysql connection configuration <br><br>

after that use this following this command:
```sh
    sql-migrate up
    go run main.go
```
